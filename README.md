# SpaghettiTools

This is a program used for cutting an image into smaller sprite and picking keycolor.<br>
The sprite information will be stored in a json file with format:

{<br>
  KeyColor: {<br>
    Red: <br>
    Green: <br>
    Blue: <br>
  }<br>
  Sprites: [<br>
    [ <x>, <y>, <width>, <height> ]<br>
  ]<br>
}<br>
